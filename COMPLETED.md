## 🎮 Управление плеером

### Клавиатура
```
Space   → Play/Pause
→ / ←   → Перемотка ±10 сек
↑ / ↓   → Громкость
M       → Mute
F       → Fullscreen
N / P   → Следующая/Предыдущая серия
Esc     → Закрыть меню
```

### Мышь/Сенсор
- Клик на озвучку → выбор озвучки
- Клик на список → выбор серии
- Клик на шестеренку → скорость 0.5x-2x
- Клик на видео → Play/Pause
- Клик на ползунок → перемотка
