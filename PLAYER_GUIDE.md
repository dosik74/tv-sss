# 🎬 Видеоплеер с поддержкой озвучек и серий

## Что реализовано

Интегрирован полнофункциональный видеоплеер в приложение с поддержкой:

### ✨ Основные возможности

1. **Выбор озвучки (Audio Tracks)**
   - Автоматическая загрузка доступных озвучек с парсера
   - Переключение между озвучками прямо в плеере
   - Меню озвучек в нижней панели контролов

2. **Выбор серий (Episode Selection)**
   - Полный список всех доступных серий
   - Отображение названия каждой серии
   - Индикатор прогресса просмотра для каждой серии
   - Быстрое переключение между сериями
   - Сохранение позиции просмотра для каждой серии

3. **Плеер**
   - HLS.js потоковое видео
   - Адаптивная доставка видео в зависимости от скорости интернета
   - Поддержка полноэкранного режима
   - Отображение текущего времени и общей длительности

### 🎮 Управление

#### Мышь / Сенсор
- **Клик на видео** - Пауза/Воспроизведение
- **Клик на прогресс-бар** - Перемотка
- **Клик на иконку озвучки** - Выбор озвучки
- **Клик на иконку списка** - Список серий
- **Клик на иконку шестерни** - Настройки
- **Клик на иконку полноэкрана** - Полноэкранный режим

#### Клавиатура
- **Space** - Пауза / Воспроизведение
- **→** - Перемотка вперед на 10 секунд
- **←** - Перемотка назад на 10 секунд
- **↑** - Увеличить громкость
- **↓** - Уменьшить громкость
- **M** - Включить/Выключить звук
- **F** - Полноэкранный режим
- **N** - Следующая серия
- **P** - Предыдущая серия
- **Esc** - Закрыть меню/Выход

### ⚙️ Функции

1. **Скорость воспроизведения**
   - Варианты: 0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x
   - Переключение через панель настроек

2. **Громкость**
   - Слайдер громкости
   - Кнопка отключения звука
   - Клавиатурное управление (↑ / ↓)

3. **История просмотров**
   - Автоматическое сохранение позиции для каждой серии
   - Сохранение прогресса в localStorage и Firestore

## 🔧 Технические детали

### Бэкенд (FastAPI)
Новый endpoint для получения информации о сериях:
```
GET /episodes?id={id}&id_type={idType}
```

Ответ:
```json
{
  "episodes": [
    {
      "number": 1,
      "title": "Название серии",
      "duration": "24 мин"
    },
    ...
  ]
}
```

### Фронтенд (React)
- **Компонент:** `VideoPlayer.tsx`
- **Функции API:** `fetchEpisodes()`, `fetchPlayUrl()`, `fetchTranslationOptions()`
- **Управление состоянием:** React Hooks (useState, useEffect, useCallback, useRef)
- **Видео:** HLS.js для потокового воспроизведения

## 🚀 Использование

### Запуск приложения

```bash
# 1. Установить зависимости (если не установлены)
npm install

# 2. Запустить dev сервер
npm run dev

# 3. В отдельном терминале запустить backend
cd server
python app.py
```

### Просмотр видео

1. Откройте приложение
2. Выберите аниме из каталога
3. Нажмите кнопку Play
4. Плеер откроется в полноэкранном режиме
5. Используйте кнопки управления для выбора озвучки, серии и управления воспроизведением

## 🐛 Возможные проблемы и решения

### Видео не загружается
- Убедитесь, что backend (FastAPI) запущен
- Проверьте консоль браузера на ошибки (F12)
- Убедитесь, что озвучка доступна для выбранного контента

### Озвучки не видны
- Проверьте, что контент поддерживает множество озвучек в Kodik
- Попробуйте другое аниме

### Серии не загружаются
- Убедитесь, что backend вернул данные об эпизодах
- Проверьте network tab в DevTools

## 📝 Примечания

- Плеер автоматически сохраняет позицию просмотра для каждой серии
- При выборе новой озвучки видео перезагружается с новыми параметрами
- Все управление интуитивно и работает как на ПК, так и на TV-пультах
- Поддержка Touch-событий для мобильных устройств
