# 🚀 Быстрый старт с новым плеером

## За 2 минуты до просмотра

### Шаг 1: Установка зависимостей
```bash
cd c:\Users\traba\Downloads\anime-cinema-tv(1)
npm install
```

### Шаг 2: Запуск фронтенда
```bash
npm run dev
```
Откроется http://localhost:5173

### Шаг 3: Запуск бэкенда (в новом терминале)
```bash
cd server
pip install -r requirements.txt
python app.py
```
Бэкенд будет на http://localhost:8000

### Шаг 4: Смотрим видео!
1. Откройте http://localhost:5173 в браузере
2. Найдите аниме в каталоге
3. Нажмите **Play**
4. Наслаждайтесь! 🎬

## Управление плеером

### На клавиатуре
| Кнопка | Действие |
|--------|----------|
| `Space` | Play/Pause |
| `→` `←` | ±10 сек |
| `↑` `↓` | Громкость |
| `M` | Mute |
| `F` | Fullscreen |
| `N` `P` | Следующая/Предыдущая серия |
| `Esc` | Закрыть |

### Мышью
- **Клик на озвучку** - выбрать озвучку
- **Клик на список** - выбрать серию
- **Клик на шестеренку** - скорость воспроизведения
- **Клик на видео** - Play/Pause
- **Клик на ползунок** - перемотка

## Если что-то не работает

### Видео не загружается
```
✓ Проверьте, что backend запущен (http://localhost:8000/health)
✓ F12 → Console → ищите красные ошибки
✓ F12 → Network → ищите запросы к /play
```

### Озвучек не видно
```
✓ Не все аниме имеют несколько озвучек
✓ Попробуйте другое аниме
✓ Проверьте /episodes endpoint в Network tab
```

### Серий нет
```
✓ Это может быть фильм (1 серия)
✓ Проверьте console на ошибки
✓ Убедитесь, что ID контента правильный
```
